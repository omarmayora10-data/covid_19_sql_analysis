**COVID-19 SQL Data Analysis Project**

***Project Overview***



This project demonstrates SQL data analysis skills using a simulated COVID-19 patient dataset.

The goal is to showcase progressive SQL proficiency, starting from basic data exploration and moving toward advanced analytical queries using SQL Server.



The project is structured intentionally for recruiters and hiring managers to evaluate:



SQL fundamentals



Analytical thinking



Use of aggregation and window functions



Real-world problem solving



***Dataset Description***



Table Name: covid\_19



Column Name	Data Type	Description

Age	INT	Patient age

Gender	VARCHAR	Patient gender

Fever	FLOAT	Body temperature (°F)

Cough	VARCHAR	Cough severity (Mild / Strong)

City	VARCHAR	Patient city

Has\_Covid	VARCHAR	COVID test result (Yes / No)



The dataset contains anonymized patient-level data across multiple Indian cities.



***Project Objectives***



Explore and clean data using SQL



Perform aggregations and conditional analysis



Apply window functions for ranking and running totals



Analyze COVID positivity patterns by demographics and symptoms



Write clean, readable, and interview-ready SQL queries
